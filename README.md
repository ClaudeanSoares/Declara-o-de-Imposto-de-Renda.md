# 🏦 Organizador de Declaração de Imposto de Renda - Bootcamp Santander

## 📝 Introdução
Este projeto nasceu da oportunidade proporcionada pelo bootcamp do **Banco Santander em parceria com a DIO.me**. O objetivo principal é consolidar os conceitos de automação financeira, migrando e evoluindo lógicas de planilhas convencionais para o ecossistema Excel.

Embora o sistema esteja em constante evolução ("não está perfeito, mas melhorar é uma questão de tempo e empenho"), ele já reflete a aplicação prática de temas como:
* Estruturação de dados fiscais.
* Interface de usuário (Menus e Formulários).
* Automação de alinhamentos e formatações.
* Implementação de equações matemáticas para cálculo de base tributável, dentre outros temas.

## 🛠️ Arquitetura e Funcionalidades

### 🧩 Script de Alinhamento e UI
Desenvolvi um motor de interface que cria um **Menu Personalizado** diretamente na barra do Google Sheets e Microsoft Excel. Isso permite que o usuário navegue entre o cadastro do **Titular**, a inserção de **Informes** e a geração de **Telas de Notas**.

### 🔢 Motor de Cálculos (Equações Matemáticas)
O script processa automaticamente as deduções e rendimentos, aplicando a lógica matemática necessária para identificar a base de cálculo, replicando as funções complexas que inicialmente estudei no Excel/VBA.

### 🎨 Formatações Personalizadas (Toques Finais)
Utilizei o script para garantir a identidade visual do projeto, aplicando cores, congelamento de linhas e ajustes de colunas de forma programática, eliminando o erro no design da planilha.
